# API Reference

This section provides detailed API documentation for Miden Crypto. For comprehensive API documentation, see the [Rust documentation](https://docs.rs/miden-crypto).

## Module Structure

- **aead**: Authenticated encryption schemes
- **dsa**: Digital signature algorithms
- **ecdh**: Elliptic curve Diffie-Hellman key exchange
- **hash**: Cryptographic hash functions
- **ies**: Integrated Encryption Scheme
- **merkle**: Merkle tree data structures
- **rand**: Random number generation
- **stark**: STARK proving system components
- **word**: Word type and utilities
