# Contributing Guidelines

Thank you for your interest in contributing to Miden Crypto!

## Getting Started

1. Fork the repository
2. Create a feature branch
3. Make your changes
4. Add tests
5. Submit a pull request

## Code Style

- Follow Rust standard formatting (`cargo fmt`)
- Run clippy (`cargo clippy`)
- Ensure all tests pass (`cargo test`)

## Documentation

- Update relevant documentation when adding features
- Add doc comments to public APIs
- Include examples in documentation

## Testing

- Add tests for new functionality
- Ensure existing tests still pass
- Consider property-based testing for cryptographic code

## Security

- Cryptographic code requires careful review
- Follow best practices for secret handling
- Use constant-time operations where appropriate

## Questions?

Open an issue or reach out to the maintainers.
