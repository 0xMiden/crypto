Expand description

Extension trait for builders that carry additional runtime context.

Some AIRs need access to data that is only available at proving time, such as bus randomness, challenge values, or witness hints. This trait lets the builder carry that data so the AIR can read it during constraint evaluation.

Existing AIRs that do not need extra context are unaffected. Only AIRs that explicitly bound on this trait will use it.
