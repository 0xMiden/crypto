1//! # Miden AIR Super Traits
2//!
3//! This crate provides unified super traits for defining AIRs (Algebraic Intermediate Representations)
4//! in the Miden VM ecosystem. Instead of implementing multiple individual traits from `p3-air`,
5//! implementers only need to implement these super traits which bundle all necessary functionality.
6//!
7//! ## Core Traits
8//!
9//! - [`MidenAirBuilder`]: Super trait for all AIR builders, containing all builder methods
10//! - [`MidenAir`]: Super trait for all AIR definitions, containing all AIR methods
11//!
12//! ## Design Philosophy
13//!
14//! The goal is to provide single traits to implement. Implementers should only work with
15//! `MidenAir` and `MidenAirBuilder`, without needing to understand or import the underlying
16//! p3-air traits.
17//!
18//! ## Usage
19//!
20//! ### Defining an AIR
21//!
22//! ```rust,ignore
23//! use miden_air::{MidenAir, MidenAirBuilder};
24//! use p3_field::{Field, ExtensionField};
25//!
26//! struct MyAir { width: usize }
27//!
28//! impl<F: Field, EF: ExtensionField<F>> MidenAir<F, EF> for MyAir {
29//!     fn width(&self) -> usize { self.width }
30//!     fn eval<AB: MidenAirBuilder<F = F, EF = EF>>(&self, builder: &mut AB) {
31//!         // All builder methods available directly on builder
32//!         let main = builder.main();
33//!         builder.when_first_row().assert_zero(main.row_slice(0).unwrap()[0]);
34//!     }
35//! }
36//!
37//! // Use macro to implement p3-air traits
38//! impl_p3_air_traits!(MyAir, BinomialExtensionField<_, 2>);
39//! ```
