Provided Methods§

Source

fn mul_w<A: Algebra<MontyField31<Self>>>(a: A) -> A

Multiply a field element (or packed field element) by W.

Defaults to standard multiplication but this can be reimplemented to make use of the exact value of W. E.g. if W = 2, 3 this should be reimplemented using addition.
